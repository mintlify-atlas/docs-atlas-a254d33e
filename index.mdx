## What is NexPhisher?

NexPhisher is a Bash-based advanced phishing tool designed for security researchers and penetration testers. It provides realistic login page replicas for over 30 popular platforms, combined with flexible port forwarding options to deliver phishing URLs via external tunneling services.

<Warning>
  NexPhisher is intended strictly for authorized security testing and educational purposes only. Unauthorized use against systems or individuals without explicit permission is illegal and unethical. Always obtain written authorization before conducting any security tests.
</Warning>

<Note>
  This project is no longer actively maintained. The successor project is [ZPhisher](https://github.com/htr-tech/zphisher), which includes updated templates and additional features.
</Note>

## Getting Started

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone git://github.com/htr-tech/nexphisher.git
    cd nexphisher
    ```
  </Step>
  <Step title="Run the setup script">
    On Linux: `bash setup` — On Termux: `bash tmux_setup`
  </Step>
  <Step title="Launch NexPhisher">
    ```bash
    bash nexphisher
    ```
  </Step>
</Steps>

See the [Installation guide](/installation) for detailed instructions, or jump straight to the [Quickstart](/quickstart).
